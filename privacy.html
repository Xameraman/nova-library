Privacy Policy for NOVA (Nexus Of Verified Assets)

Last updated: July 4, 2026

1. What data does NOVA collect?

NOVA only processes the information needed to run its commands and keep the library functional.

  • Slash commands – when you use a command like /search, /nova-add, or /share-with-nova, NOVA sees your Discord user tag and the options you provide (pack names, links, categories, etc.).
  • Temporary search sessions – when you run a search or browse the library inside Discord, NOVA keeps a short‑lived session in memory (your user ID and the search results). This session is automatically deleted after 60 seconds of inactivity.
  • Command logs – NOVA logs who used a staff command and which pack was affected (e.g., “@User used /nova-add — Pack: Example Pack”). These logs are sent to a private staff‑only channel and are never shared publicly.
  • User submissions – if you submit a pack, report, or idea, NOVA records your Discord tag, the pack name, and any attached proof images. This information is only visible to staff in dedicated private channels.

2. What data does NOVA NOT collect?

  • NOVA does not read, store, or log the content of your private messages or any message that is not a direct slash command.
  • NOVA does not collect email addresses, IP addresses, or any personal identifiers outside of the Discord username/tag provided by the platform.
  • NOVA does not sell, trade, or share your data with any third party.

3. How is data stored and protected?

  • Pack data (the library itself) is stored in a private Google Sheet and synced to GitHub as JSON files. This data contains pack names, creator names, download links, and view counts — no personal user information.
  • Temporary data (search sessions) is held only in the bot’s memory and wiped after 60 seconds.
  • Logs and submissions are stored in private Discord channels accessible only to authorised staff members.

4. Data retention and removal

  • Temporary search sessions are automatically deleted after 60 seconds.
  • Command logs and submissions are kept indefinitely for moderation and record‑keeping purposes. If you would like your data removed from these logs, you can request deletion by contacting the server owner. Deletion will be carried out within a reasonable timeframe.
  • Pack data (library entries) does not contain personal user information and is retained as part of the public library.

5. Children’s Privacy

NOVA is not directed at children under the age of 13. We do not knowingly collect information from children. If you believe a child has provided personal data through the bot, please contact the server owner and we will take steps to delete it.

6. Changes to this policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the bot’s documentation or the Discord server where the bot operates.
